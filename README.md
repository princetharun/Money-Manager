# 💰 Money Manager

A personal finance tracker web app — built with plain HTML, Firebase \& Chart.js. No frameworks, no installs. Just open and use!

## ✨ Features

* 🔐 **Login / Signup** — Firebase Authentication
* 📅 **Monthly Budget** — Set budget per month, auto-carries forward
* ➕ **Daily Entries** — Add income \& expense with description + date
* ⚡ **Quick Add** — One-tap FAB button for fast expense entry with category chips
* ⚠️ **Budget Alerts** — Auto warning when 80% / 90% / 100% of budget is spent
* 📊 **Graphs** — Daily income vs expense bar chart + Monthly overview chart
* 🧾 **Receipts** — Weekly grouped receipts with PDF download
* 🎯 **Savings Goals** — Set goals with emoji, track progress
* 🌙 **Dark Mode** — Full dark theme support
* 📲 **PWA** — Install as app on mobile (Add to Home Screen)
* 💾 **Export CSV** — Download month data as CSV
* 🗑️ **Safe Reset** — Type "RESET" to confirm before deleting data

## 🔄 Update / Deploy

To update the app:

1. Edit `index.html` locally
2. Go to your GitHub repository
3. Click `index.html` → Upload new file (same name)
4. Commit changes → Auto deploys in \~2 minutes ✅
